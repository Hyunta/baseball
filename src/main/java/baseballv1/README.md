## Outlook:

- 컴퓨터는 1~9까지 서로 다른 임의의 수 3개를 선택
- 컴퓨터는 플레이어가 입력한 숫자와 비교해서 결과출력
- 모두 다를 경우 Out
- 이와 같은 과정을 정답을 맞추면 종료
- 종료된 후 다시 시작 or 완전히 종료
